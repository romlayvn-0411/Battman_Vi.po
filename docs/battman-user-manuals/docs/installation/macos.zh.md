# 在 macOS 上安装

目前 **尚未提供 Battman 的 macOS 原生界面**，macOS 支持仍在规划中。

当前 Battman 主要面向越狱设备或通过 TrollStore 安装的 iOS 设备。一些底层组件和库可以在 macOS / Apple 芯片 Mac 上编译用于开发和调试，但暂时没有面向普通用户的 macOS 图形界面应用。

## 当前状态

- 暂无官方的 macOS 应用包或安装器。
- 某些核心库可以在 macOS 11+ 上编译，仅供开发使用。
- 与电池、温度相关的 macOS API 仍在调研中。

## 当 macOS 版本准备就绪时

本页将补充说明：

- 系统要求（支持的 macOS 版本与机型）。
- 如何获取并安装 Battman for macOS（下载、校验与运行方式）。
- 沙盒、权限与数据目录的注意事项。

在此之前，如需实际使用，请参考 iOS 安装章节。
